How it works
01

Upload or Select Agent

Point A-DAP-T at a demo agent or upload your own AI-agent project ZIP. No code is executed — analysis is fully static.

02

Static + Rule-Based Scan

A-DAP-T parses prompts, tool configs, and code for known risk patterns across six categories — no runtime required.

03

Get a Risk Report

Receive a safety score, per-finding explanations, file references, and a remediation checklist you can act on immediately.

Risk categories
Prompt Injection Risk

Unguarded inputs that let users override agent instructions

Secret Exposure Risk

API keys, tokens, or credentials hardcoded in prompts or config

Tool Permission Risk

Tools with overly broad access — delete, write, or send without scope limits

Human Approval Risk

High-stakes actions that execute without human-in-the-loop confirmation

Data Exposure Risk

PII, internal data, or sensitive context sent to external models or APIs

Auditability Risk

Agents with no logging, traceability, or audit trail for actions taken
